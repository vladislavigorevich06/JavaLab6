Аннотация @ToString

Аннотация для указания, какие поля включать в строковое представление объекта.

Применение

  • Поля с Mode.YES будут отображаться в toCustomString().
  • Поля с Mode.NO будут скрыты.

Поля

value(): Mode — режим отображения поля (YES/NO)

Примеры

@ToString(Mode.YES)
private String name;

@ToString(Mode.NO)
private String password;
