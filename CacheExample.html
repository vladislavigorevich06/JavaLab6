Класс ToStringExample

Пример использования аннотации @ToString для управления строковым представлением объекта.

Конструктор

public ToStringExample(String name, String password, int age, String email)

Инициализация полей объекта.

Методы

  • getName() — возвращает имя
  • getPassword() — возвращает пароль
  • getAge() — возвращает возраст
  • getEmail() — возвращает email
  • isPasswordHidden() — проверяет, скрыт ли пароль
  • toCustomString() — формирует строковое представление объекта с учетом аннотации

Примеры использования

ToStringExample obj = new ToStringExample("Alice", "12345", 25, "a@b.com");
System.out.println(obj.toCustomString());
