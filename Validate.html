Аннотация @Validate

Аннотация для проверки корректности типов классов.

Применение

Указывается массив классов, которые будут валидироваться:

@Validate({String.class, Integer.class, Double.class})
public class ValidateExample { ... }

Назначение

  • Проверка входных данных на соответствие допустимым типам.
  • Обработка ошибок при передаче неверного класса.
