Аннотация @Cache

Аннотация для указания кешируемых областей объекта.

Применение

Список кешируемых сущностей указывается через параметр value:

@Cache({"users", "products", "orders"})
public class CacheExample { ... }

Назначение

  • Упрощение доступа к кешируемым областям.
  • Автоматическое создание кеша для указанных областей.
